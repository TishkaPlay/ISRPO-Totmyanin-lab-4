## Описание проекта

Данный проект представляет собой комплексную лабораторную работу по закреплению навыков работы с Git, Markdown, терминалом и созданию структуры проекта. Включает в себя frontend часть (HTML-страницы), backend часть на C# (.NET), практику работы с терминалом и демонстрацию возможностей Markdown.
